Roadmap

Eclipse Enhancement 344422 tracks the possible future inclusion of InTrace with the Java Development Tools (JDT) project.

Target for Release 20

  • Refactor UI
    • Merge Connection/Instrumentation tabs
    • Add Settings Window to contain Trace/Advanced tabs
    • Move all output actions to one row with the following grouping:
      • Collect Trace (checkbox), Save (icon), Clear(icon)
      • Filter (checkbox), Filter... (Text)
      • Align auto scroll to the right to be closer to the scrollbar
    • Add confirmation of clearing output
  • Add some UI tests and more UTs

Target for Future Releases

  • Exit trace in finally block
  • GUI Agent Loader
  • VisualVM
    • Productise integration
    • Launch client with more memory
