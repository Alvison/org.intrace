Eclipse

The quickest way to use InTrace is to launch it from within Eclipse. To install the Eclipse plugin you can use the following update site. 

http://intracesite.appspot.com/

Full details: InTrace Eclipse

Standalone

InTrace can be used with any Java process along with the standalone UI. To do this you will need to download the following two files.

  • intrace-agent.jar
  • intrace-ui.jar
